ALPrivacy and reliability
Built for local operation

Client confidence

Hosted inside the club, secured like a serious business system.

Local network hosting

The server runs on a Windows machine inside the club. Staff connect through the LAN using a browser or installed PWA shortcut.

Encrypted ID images

Stored ID images are protected with AES-256 encryption, reducing exposure if files are accessed outside the application.

Session protection

JWT authentication and protected API routes ensure users must be logged in and authorized for each action.
